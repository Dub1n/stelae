# Starter Bundle

This bundle installs the fetch/memory/strata/scrapling/filesystem/rg/shell helpers and publishes their catalog overrides.

Install via `python scripts/install_stelae_bundle.py` or point the installer at this folder. The installer copies the folder to `${STELAE_CONFIG_HOME}/bundles/starter/`, registers install refs, and triggers `manage_stelae install_server` for each server descriptor.
